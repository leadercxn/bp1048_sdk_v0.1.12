clk_example工程主要演示了BP10时钟配置选择，系统时钟参考源选择。

	1: 24M晶体起振DPLL锁定288M，内核时钟为PLL时钟，系统时钟PLL 2分频为144M。
	2: 24M晶体起振APLL锁定240M，内核时钟为PLL时钟，系统时钟PLL 2分频为120M。
	3：RC12M，系统工作于RC时钟下

注意：
	选择对应输入时钟源时，如果是外部晶体则需要焊接相关晶体


硬件环境要求:
    1. BP10系列开发板
    2. 外接串口小板，TX/RX/GND 接至 A25/A24/GND 引脚 (波特率 256000)
    3. GPIOA0输出PWM波形，PWM频率为系统时钟10分频，可以接示波器观察PWM波形频率，推算系统时钟。

软件使用说明:
	上电启动之后，系统默认RC起振，然后选择24M晶体锁定DPLL288M，APLL240M配置UART相关IO，并从GPIOA23上输出PWM波形；
	1. UART接受到电脑串口发下的命令'A'，切换到24M晶体，DPLL锁定288M，系统工作于144M。
	2. UART接受到电脑串口发下的命令'B'，切换到24M晶体，APLL锁定240M，系统工作于120M。
	3. UART接受到电脑串口发下的命令'C'，切换到12MRC时钟，系统工作于12M RC时钟。RC时钟有5%的偏差。
	
说明：
	1、BP10系统输入晶体支持无源晶体和有源晶体
	2、BP10系统支持的晶体频率范围：1M，2M，...，40MHz
