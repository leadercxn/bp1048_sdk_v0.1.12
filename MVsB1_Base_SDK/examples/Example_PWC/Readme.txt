Example_PWC工程主要演示了 PWC的工作流程。

硬件环境要求:
    1. BP10系列开发板；
    2. 外接串口小板，TX/RX/GND 接至 A25/A24/GND 引脚 (波特率 256000)；
    3. 上位机串口调试工具
    4. 示波器

软件使用说明:
	上电启动之后，芯片初始化；	
	通过串口可以看到PWC工作模式选择,通过输入数字1~3选择不同的模式，如：
	-------------------------------------------------------
	上电启动之后，可通过串口输入不同指令，进入例程；
	1. 输入‘1’，PWC单次捕获模式例程；
	2. 输入‘2’，PWC连续捕获模式例程；
	3. 输入‘3’，PWC中断模式例程；
	-------------------------------------------------------
