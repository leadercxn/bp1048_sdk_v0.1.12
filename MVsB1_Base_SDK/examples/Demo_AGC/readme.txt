Demo_AGC工程主要演示了mic输入到DAC输出 左声道启用了AGC自动增益控制。
示例程序代码运行于 BP10xx系列开发板。

功能：
1.MIC1，MIC2输入，AGC调整增益，DAC输出。
2.上电后，mic附近播放不同音量的音乐或语音，可观察到log会根据输入音量自动调节增益。右声道配置mic的0 db 输出。左声道使用了AGC，输出音量相对稳定。

硬件环境要求:
    1. BP10xx系列开发板
    2. 外接串口小板，TX/RX/GND 接至 A25/A24/GND 引脚 (波特率 256000)；

优化说明：
1、Target level和Max level差值不能小于3，过小会引起增益调整的震荡，输入人声和音乐对这两个值设置有一定差异。
2、Attack step time配置可缩小，提高限幅响应速度。 
3、Frame time 这个值可配合软件系统帧大小进行设置。
