ADC_Example工程主要演示了BP10系列SarADC采样相关示例。

硬件环境要求:
    1. BP10系列开发板
    2. 外接串口小板，TX/RX/GND 接至 A25/A24/GND 引脚 (波特率 256000)；
    3. GPIOA26为演示功能默认输入模拟管脚。

软件使用说明:
    1. 上电启动之后，通过串口接收指令，进入相应的电压采样case，并通过UART输出；
    2. 输入'a'，采样GPIOA26端口相对电压
    3. 输入'b'，采样LDOIN的绝对电压，串口打印的是电压转换值，单位mV
    4. 输入'c'，采样PowerKey端口的相对电压

说明：
    1、注意需要选择参考电压源和模块工作时钟频率。
    2、注意GPIOA26，PowerKey在所在的Demo板或者开发板是否有相应的管教，如果没有请更换测试管教。
