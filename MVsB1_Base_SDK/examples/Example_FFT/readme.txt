FFT_Example工程主要演示了FFT的基本使用方法。

功能：
验证RFFT，RIFFT, CFFT和CIFFT是否正确。


硬件环境要求:
    1. 使用BPxx系列开发板时，TX/RX/GND 接至 A25/A24/GND 引脚 (波特率 256000)；

软件使用说明:
    上电启动之后，会打印如下信息，这时输入‘r’用于示例RFFT/RIFFT使用，输入‘c’用于示例CFFT/CIFFT如何使用。如果打印信息均为“Ok”则证明功能正常。
