SPDIF_Example工程主要演示了SPDIF的基本使用方法。

功能：
1.演示SPDIF输入，I2S输出过程；
2.演示PCM数据通过SPDIF输出过程。

硬件环境要求:
    1. BP10系列开发板；
    2. 第三方光纤发送设备和第三方光纤接收设备
    3. 使用BP10系列开发板时，外接串口小板，TX/RX/GND 接至 A25/A24/GND 引脚 (波特率 256000)；

软件使用说明:
    上电启动之后，可通过串口输入不同指令，进入3种例程；
    1. 输入‘R’，进入SPDIF接收模式；
    2. 输入‘T’，进入SPDIF发送模式。
    3. 输入‘A’，进入ADC输入SPDIF发送模式。
